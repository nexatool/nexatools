What Is PDF Merging?

Merging PDFs means combining two or more separate PDF documents into a single file, with pages appearing in the order you choose. Instead of sending five separate attachments, you can package everything into one clean document that opens with a single click.

This is especially useful when documents are related for example, a cover letter and a resume, several scanned receipts, or chapters of a report and need to travel together as one file rather than several loose ones.

How Your Privacy Is Protected

When you upload PDFs into the tool above, they are read directly by your browser and combined using JavaScript running on your device. At no point are your files sent across the internet to a remote server or stored in any database. Once you close or refresh the tab, the files and the merged result are simply gone from memory, so there's nothing left behind for anyone to access.

Common Problems & Troubleshooting

1. "Merge PDFs" Button Stays Disabled

This button only activates once at least two PDF files have been uploaded. If it stays greyed out, check that both files appear in the list above.

2. "PDF Engine Failed to Load" Error

This means the merging library couldn't load in your browser. Check your internet connection and refresh the page, then try again.

3. Merge Fails or Shows an Error

This usually happens when one of the uploaded files is corrupted or password-protected. Remove the password from the affected PDF using a PDF editor, then try merging again.

4. File Was Skipped on Upload

Files larger than 25 MB or files that aren't valid PDFs are skipped automatically, and a warning message will explain why.

5. Files Merged in the Wrong Order

Drag and drop each file in the list to reorder them before clicking "Merge PDFs" the order number on each file shows exactly where it will appear in the final document.
