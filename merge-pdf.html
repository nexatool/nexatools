Tips for Getting the Best Results

Merging PDFs is straightforward, but a few tips help you get a clean final document:

1. Check the Order Before Merging

Drag your files into the list order you want the pages to appear in the final PDF matches the order shown on screen.

2. Remove Unwanted Files First

Use the remove button on any file card to take it out of the merge before combining, no need to start over.

3. Password-Protected PDFs Aren't Supported

If a PDF is encrypted or password protected, remove the password first using a PDF editor, then merge it here.

4. Keep Total File Size Reasonable

Merging many large PDFs at once can be slower since everything runs locally in your browser. Merge in smaller batches if needed.
