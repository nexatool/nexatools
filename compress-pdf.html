Tips for Getting the Best Results

Compressing a PDF is straightforward, but a few tips help you get the smallest possible file:

1. Start with Medium

Medium compression gives a good balance of quality and size for most documents try it first.

2. Use High for Sharing by Email

If you need the smallest possible attachment size, choose High compression before sending.

3. Use Low for Print-Quality Documents

If your PDF will be printed or contains detailed images, Low compression preserves the most quality.

4. Password-Protected PDFs Aren't Supported

If a PDF is encrypted or password protected, remove the password first using a PDF editor, then compress it here.
