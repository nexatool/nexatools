Tips for Getting the Best Results

1. Choose the Right Level

Use Low if the PDF will be printed or needs the sharpest possible detail. Medium works for most everyday sharing. High is best when you just need the smallest possible file, such as for email attachments.

2. Best for Scanned Documents and Image-Heavy PDFs

Compression here gives the biggest savings on scanned pages, photo-heavy reports, and design files with large images.

3. Text-Based PDFs Lose Selectable Text

Since each page becomes an image, text in the compressed file is no longer selectable or searchable. If you need to keep editable text, compress a copy instead of your original.

4. Password-Protected PDFs Aren't Supported

If a PDF is encrypted or password protected, remove the password first using a PDF editor, then compress it here.

Common Problems & Troubleshooting

1. "Compress PDF" Button Stays Disabled

This button only activates once a valid PDF has finished loading into the tool. If it stays greyed out, check that your file appears in the upload area above and that the compression level selector has appeared.

2. "PDF Engine Failed to Load" Error

This means the compression libraries couldn't load in your browser. Check your internet connection and refresh the page, then try again.

3. Compression Fails or Shows an Error

This usually happens when the PDF is corrupted or password-protected. Remove the password using a PDF editor first, then try compressing again.

4. File Was Rejected on Upload

Files larger than 25 MB or files that aren't valid PDFs are rejected automatically, and a warning message will explain why.

5. Compressed File Doesn't Look as Small as Expected

Try a higher compression level (Medium or High) for smaller output, or note that already-optimized text-based PDFs may not shrink as dramatically as scanned image-heavy files.

6. Text Is No Longer Selectable After Compressing

This is expected — each page is rasterized into an image during compression. If you need to keep selectable text, compress a copy and keep your original PDF for editing.
