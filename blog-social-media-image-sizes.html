Best Image Sizes for Facebook, Instagram, LinkedIn & X in 2026

📅 July 11, 2026 ⏱ 9 min read ✍️ NexaTools Team

Every social platform crops, resizes, and compresses images differently, and uploading the wrong dimensions is one of the fastest ways to make a professional-looking post look amateurish. A profile photo that gets cropped mid-face, a cover banner with text cut off at the edges, or a blurry post image are all usually the result of using the wrong size, not bad design.

This cheat sheet covers the current recommended image dimensions for Facebook, Instagram, LinkedIn, and X (formerly Twitter) as of 2026, along with practical tips for keeping your images sharp on every platform.

Facebook Image Sizes

Profile Picture
180 x 180 pixels for display, though Facebook recommends uploading at least 320 x 320 pixels so it stays sharp on high-resolution screens.
Cover Photo
820 x 312 pixels on desktop, 640 x 360 on mobile. Keep important text and faces centered, since edges are cropped differently across devices.
Shared Post Image
1200 x 630 pixels is the standard for link shares and single-image posts, matching a roughly 1.91:1 ratio.
Facebook Story
1080 x 1920 pixels, a full-screen vertical 9:16 ratio.

Instagram Image Sizes

Profile Picture
320 x 320 pixels, displayed as a circle so keep important detail away from the corners.
Square Post
1080 x 1080 pixels, the classic 1:1 ratio and still the safest choice for feed consistency.
Portrait Post
1080 x 1350 pixels (4:5 ratio) takes up more vertical space in the feed, which often increases visibility.
Story & Reels
1080 x 1920 pixels, full-screen vertical 9:16 ratio, same as Facebook Stories.

LinkedIn Image Sizes

Profile Picture
400 x 400 pixels minimum recommended for a crisp display across desktop and mobile.
Cover Banner
1584 x 396 pixels for personal profiles. Company pages use a slightly different banner size of 1128 x 191 pixels.
Shared Post Image
1200 x 627 pixels works well for both single images and link preview shares.

X (Twitter) Image Sizes

Profile Picture
400 x 400 pixels, displayed as a circle similar to Instagram.
Header Photo
1500 x 500 pixels for the banner across the top of a profile.
In-Feed Image
1200 x 675 pixels (16:9 ratio) displays cleanly in the timeline without excessive cropping.

File Format and Size Considerations

Beyond pixel dimensions, file format also affects how your image looks after upload. JPG is generally best for photos, while PNG is better for graphics with text, logos, or sharp edges since it avoids the soft artifacts JPG compression can introduce. If you are unsure which to use, our image format guide breaks down the differences in detail.

Most platforms also re-compress uploaded images automatically, so keeping your file size reasonable before uploading (generally under 1–2MB for feed posts) helps avoid additional quality loss during that automatic compression. Use our Image Compressor tool if your file is larger than necessary.

Frequently Asked Questions

Why does my Instagram post get cropped after uploading?

Instagram crops images that don't match its supported aspect ratios (1:1, 4:5, or 9:16). Resizing to one of these ratios before uploading prevents unexpected cropping.

What is the best image format for social media?

JPG works well for photographs due to smaller file sizes, while PNG is better for graphics, logos, and text-heavy images where sharp edges matter.

Do social platforms compress uploaded images?

Yes, nearly all major platforms automatically compress images after upload to save bandwidth. Uploading a reasonably sized, well-compressed image to begin with reduces additional quality loss.

Should I use the same image size across all platforms?

Not recommended. Each platform has different aspect ratios and cropping behavior, so resizing individually for each platform produces the cleanest results.
