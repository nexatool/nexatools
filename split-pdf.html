What Is PDF Splitting?

Splitting a PDF means pulling specific pages, or a range of pages, out of an existing PDF and saving them as a brand-new file. Instead of sending someone an entire 40-page report when they only need page 12, you can extract exactly the pages that matter and share a much smaller, focused document.

This is different from compressing a PDF, which reduces file size while keeping every page. Splitting changes which pages are included, letting you reshape a document into exactly the subset you need.

Tips for Getting the Best Results

Splitting a PDF is straightforward, but a few tips help you get exactly the pages you need:

1. Check the Total Page Count First

Once your file is uploaded, the tool shows how many pages it contains. Use this to plan your range before extracting.

2. Use Commas for Non-Consecutive Pages

You can mix single pages and ranges freely, for example 2,4,7-9, and every match will be included in the extracted PDF.

3. Order Matters

Pages are extracted in the order you list them, so 5,1-3 will place page 5 first, followed by pages 1 through 3.

4. Password-Protected PDFs Aren't Supported

If a PDF is encrypted or password protected, remove the password first using a PDF editor, then split it here.

Common Problems & Troubleshooting

Running into an issue while splitting your PDF? Here are fixes for the most common problems:

⚠️ "This file could not be read as a PDF"

The file may be corrupted or password protected. Remove any password protection using a PDF editor first, then upload it again.

⚠️ "Please enter a valid page range"

Double-check that every number in your range falls between 1 and the total page count shown after upload, and that ranges are written as start-end (e.g. 1-3).

⚠️ File larger than 25 MB

Very large PDFs aren't supported in-browser. Try compressing the PDF first with a PDF compression tool, then split it.

⚠️ Page count not detected

If the page count badge never appears, your connection may have blocked the PDF engine from loading. Refresh the page and try again.

⚠️ Download button doesn't work

Some browsers block automatic downloads from JavaScript. Check your browser's download permissions or pop-up settings for this site.
