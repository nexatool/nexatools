How to Choose the Right PDF Tool

Each tool below solves a different, specific problem. Here's a quick guide to picking the right one for your task:

Need an editable copy of a PDF?

Use PDF to Word to turn a PDF into a document you can edit in Word.

Have a Word document to share?

Use Word to PDF to convert DOCX files into a fixed, shareable PDF.

Have several PDFs to combine?

Use Merge PDF to join multiple files into one document, in the order you choose.

Need only certain pages?

Use Split PDF to extract or separate specific pages from a larger file.

Is your file too large to send?

Use Compress PDF to shrink the file size while keeping it readable.

Still not sure? Read more about choosing the right file format in our guide on PDF vs DOCX, or browse the full tools directory for other free utilities.
